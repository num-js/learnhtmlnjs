Iframe Tag

<iframe> Tag is use to use to display a webpage within a Webpage.

Iframe Tag - Attributes

scrolling value:yes,no,auto
scrolling attribute is use to define whether or not to display scrollbars in the iframe.

frameborder value:number,pixel
frameborder attribute is use to define the border of the iframe.

name value:text
name attribute is use to define thename of the iframe.

href value:url
href attribute is use to define the url of the iframe.


Iframe - Example 

<iframe frameborder="2" name="if1" scrolling="auto" href="www.learnhtmlnjs.tk">
</iframe>
