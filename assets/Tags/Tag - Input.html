Input Tag

<input> Tag is use to use to define Unorder List in a Webpage.

Input Tag - Attributes

name: value: B1,A2(name of the input tag)
name attribute is use to define the name of the input tag .

disabled: value: -
disabled attribute is use to define that a input element should be disabled when the webpage is loaded .

value: value: submit(value of the input element)
value attribute is use to define an initial value for the input element .

autofocus: value: -
autofocus attribute is use to define that a input element should automatically get focus when the webpage is loaded .

type: value: button,checkbox,color ,date,datetime-local,email,file,hidden,image,month,number,password,radio,range,reset,search,submit, teltext,time,url,week.

type attribute is use to define the type of the input .
