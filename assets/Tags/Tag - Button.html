Button Tag

<button> Tag is use to use to Insert a Button in a Webpage.

Button Tag - Attributes

name: value: B1,A2(name of the button)
name attribute is use to define the name of the button .

disabled: value: -
disabled attribute is use to define that a button should be disabled when the webpage is loaded .

value: value: click me(value of the button)
value attribute is use to define the an initial value for the button that will appear on the button .

autofocus: value: -
autofocus attribute is use to define that a button should automatically get focus when the webpage is loaded .

type: value: submit,button,reset
type attribute is use to define the type of the Button .


Button Tag - Example 

<form>
Name:     <input type="text" name="n1"><br>
Password: <input type="text" name="n2"><br>
<button autofocus name="nu1" type="click" value="click">click</button>
<button type="submit" name="nu2" value="Submit">Submit</button>
<button type="reset" name="nu3" value="Reset">Reset</button>
</form>
